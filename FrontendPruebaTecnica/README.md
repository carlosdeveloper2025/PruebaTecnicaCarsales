# PruebaTecnicaCarsales

## Resumen
Proyecto que implementa un BFF (.NET 8) y un frontend (Angular) que consumen la API pública Rick and Morty (`/api/episode`).
Se implementa paginación, separación de lógica, manejo de errores y uso de configuración.

## Requisitos
- .NET 8 SDK
- Node.js 18+ y Angular CLI (para Angular 19)
- Git

## Ejecutar Backend
1. Ir a `backend/PruebaTecnicaCarsales.Api`
2. Restaurar y ejecutar:
   ```bash
   dotnet restore
   dotnet run
